# Goniómetro Wild G-10 — primera estructura con diseño moderno

Esta versión conserva el planteamiento de la primera web:
- generalidades;
- descripción por componentes;
- vistas generales;
- puesta en estación;
- vídeo de la lupa;
- fase de lecturas pendiente.

La interfaz se ha adaptado al último diseño, manteniendo las imágenes originales.
Para evitar confusión, las capturas antiguas están identificadas como imágenes de referencia y las acciones reales se realizan mediante botones externos.

Abra `index.html` para probarla. No requiere servidor.
